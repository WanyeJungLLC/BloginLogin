# Custom Application Components

This folder contains **8 custom components** specific to this blog application.

- `blog-list.tsx` - Blog post listing
- `hero.tsx` - Homepage hero section
- `image-upload.tsx` - Image upload functionality
- `layout.tsx` - Page layout wrapper
- `markdown-editor.tsx` - Markdown editing with preview
- `markdown-renderer.tsx` - Markdown content display
- `ObjectUploader.tsx` - Object storage uploader
- `portfolio-grid.tsx` - Portfolio items grid
