# Vendor: Replit Integrations

This folder contains **Replit-provided integration code** for object storage.

These are **library/repackaged code** - not custom application code.

## Object Storage Integration
- `object_storage/index.ts` - Main exports
- `object_storage/objectStorage.ts` - Storage service implementation
- `object_storage/objectAcl.ts` - Access control
- `object_storage/routes.ts` - API routes for file uploads

## Usage
This integration handles file uploads to Replit's object storage service with automatic bucket management.
